### `login`

Start by running `login` including your `--provider` (shorthand `-p`) of choice (`google` or `github`)

```bash [terminal]
orbiter login
> Login with OAuth

OPTIONS:
  --provider, -p <str> - Choose the OAuth provider you want to login with (github | google)

FLAGS:
  --help, -h - show help

USAGE:
orbiter login --provider google
```
