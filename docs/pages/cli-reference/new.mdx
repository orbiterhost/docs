### `new`

In just one command create a new project through an available template and deploy

```bash [terminal
orbiter new
> Create a new app ready to deploy

ARGUMENTS:
  [name] - Name of the new project [optional]

OPTIONS:
  --template, -t <str> - Optional template you want to use [optional]

FLAGS:
  --help, -h - show help

USAGE:
orbiter new --template bhvr
```
