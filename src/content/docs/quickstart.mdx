Welcome to Orbiter, the easiest way to host static websites! In this brief walkthrough we’ll show you how to launch a site in no time.

## Create an Account

Visit [app.orbiter.host](http://app.orbiter.host) and create a free account with your Google or GitHub account

## Create a New Site

Click on the New Site button in the top right and add your website file or folder. This could be:

- A single html file
- A folder with html, css, and js
- A build folder from a framework like React, Astro, or Hugo

After selecting your files to upload also select your free [Orbiter.website](http://Orbiter.website) subdomain. Then click submit!

You’re done! Told you it was easy. Now you can visit your site in all its galactic wonder.
